# Six-Sigma-Logistics-Analysis

Pareto Analysis revealed that 51.7% of all delivery defects occur in just two states: SP and RJ. This indicates a localized infrastructure bottleneck rather than a global seller issue.

Solution 1: Regional Fulfillment Centers. Based on the Pareto analysis, Olist should pilot a "Fast-Track" warehouse in Rio de Janeiro to reduce the 51% late delivery contribution.

Solution 2: Buffer Adjustment. Analysis shows that the order_estimated_delivery_date is too aggressive for interstate shipping. Increasing the buffer by 2 days for Region X would align customer expectations with carrier capability.
